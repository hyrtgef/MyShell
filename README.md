# MyShell

Простая командная оболочка.

## Возможности

- Встроенные команды: `cd`, `pwd`, `exit`, `help`, `echo`, `history`
- Запуск внешних программ: `ls`, `cat`, `grep`, `wc`
- Перенаправление: `>` (запись), `>>` (дозапись), `<` (чтение)
- Pipeline: `|` (цепочка команд)
- Фоновый запуск: `&`
- История команд

## Сборка

```bash
mkdir build && cd build
cmake ..
make

# Запуск
./myshell

## Требования
```
g++ 10+ (C++20)
CMake 3.10+
